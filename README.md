# 🎬 YouTube Downloader

Aplicação desenvolvida em Python para download de vídeos do YouTube, utilizando biblioteca de integração com a plataforma.

## Funcionalidades

- Download de vídeos do YouTube via URL
- Seleção de qualidade e formato do vídeo
- Interface simples e funcional

## Tecnologias

- **Python 3**
- **yt-dlp** (integração com YouTube)

## Como executar

1. Instale a dependência:
```bash
pip install yt-dlp
```
2. Clone este repositório:
```bash
git clone https://github.com/SEU-USUARIO/youtube-downloader.git
```
3. Execute o arquivo:
```bash
python main.py
```
